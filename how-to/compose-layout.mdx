### Add tab to under block

`tabOptions`

블록 아래에 탭을 추가합니다.

```yaml
blocks:
  - type: http
    tabOptions:
      autoload: 1
      # keep: true
      tabs:
        - name: 1
        - name: 2
```

### Use tab block alone

`type: tab`

탭 블록을 단독으로 사용합니다.

```yaml
blocks:
  - type: tab
    tabOptions:
      tabs:
        - name: 1
```

### Use toggle block

`type: toggle`

다른 블록의 내용을 접었다 피면서 볼 수 있습니다.

```yaml
blocks:
  - type: toggle
    name: Toggle
    icon: tree
    class: text-lg p-2 shadow rounded text-green-700
    toggledClass: font-medium text-green-700 bg-green-600/10
    # toggled: true
    blocks:
      - type: http
```

### Use defined layout block

`type: top | left | center | right | bottom`

빠르게 레이아웃을 구성할 수 있는 규정된 레이아웃 블록입니다.

```yaml
blocks:
  - type: top
  - type: left
  - type: center
  - type: right
  - type: bottom
```

### Set custom layout

`layout`

HTML, CSS로 커스텀 레이아웃을 설정할 수 있습니다.

```yaml
layout:
  style:
    # max-width: 1400px
    margin: 0px auto
    display: flex
    flex-wrap: wrap
  div:
    - name: page1
      style:
        width: 100%
    - name: page2-1
      style: 
        width: 50%
        padding-right: 20px
    - name: page2-2
      style: 
        width: 50%
    - name: page3
      style:
        width: 100%
blocks:
  - layout: page1
    type: markdown
  - layout: page2-1
    type: markdown
  - layout: page2-2
    type: markdown
  - layout: page3
    type: markdown
```
