### Get related data in modal with parameter

`modals`, `openModal`

- 모달로 조회

```yaml
type: http
columns:
  id:
    openModal: modal1-:id
modals:
  - path: modal1-:id
    blocks:
      - type: http
        params:
          - key: id
            valueFromRow: id
```

### Get data in tab with parameter

`tabOptions` , `tabs`

- 탭에서 조회

```yaml
type: http
tabOptions:
  tabs:
    - name: 1
      blocks:
        - type: http
          params:
            - key: id
              valueFromRow: id
```

### Reload data after updating parameter of another block

`updateParams`

- 다른 블록에서 조회(레이아웃)

```yaml
blocks:
  - type: left
    blocks:
      - type: http
        method: GET
        fetchFn: |
          return [
            { id: 1, name: 'Alpha' },
            { id: 2, name: 'Bravo' },
            { id: 3, name: 'Charlie' },
            { id: 4, name: 'Delta' },
            { id: 5, name: 'Echo' }
          ];
        columns:
          id:
            updateParams:
              detail_id: "{{id}}"
  - type: right
    blocks:
      - type: http
        method: GET
        fetchFn: |
          if (!detail_id) return [];

          return [{ id: detail_id, name: `detail-${detail_id}` }];
        params:
          - key: detail_id
```

### Send parameter to another page

`valueFromSearch`

페이지 이동해서 조회

```yaml
columns:
  id:
    openPage: value-from-search-:id

---
pages:
  - path: value-from-search-:id
    blocks:
      - type: http
        params:
          - key: id
            valueFromSearch: id
```
