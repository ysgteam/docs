---
title: "Run actions"
description: "버튼을 눌러 액션을 실행하는 방법들입니다."
---

### Add action button

액션 버튼 추가하기

```yaml
blocks:
  - type: http
    actions:
      - label: 실행
        # type: http
```

### Set placement of action button

액션 버튼 위치를 조정합니다.

```yaml
blocks:
  - type: http
    actions:
      - label: 실행
        placement: top right
        # placement: left bottom
```

### Run action in single

단독 실행 액션

```yaml
actions:
  - type: http
    single: true
```

### Run action with selected rows

테이블의 여러 내역을 선택한 다음 액션 실행하기

```yaml
actions:
  - type: http
    # forEach: true # 선택 행마다 순차 실행
selectOptions:
  enabled: true
```
