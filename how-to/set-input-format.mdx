---
title: "Set input format"
description: "입력 필드에 사용할 수 있는 다양한 사전정의 포맷을 제공합니다."
---

### Textarea

```yaml
params:
  - key: memo
    format: textarea
```

### Number

```yaml
params:
  - key: rating
    format: number
```

### Date

```yaml
params:
  - key: open_date
    format: date
```

### Color

```yaml
params:
  - key: color
    format: color
```
