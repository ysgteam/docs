### Basic form

```yaml
blocks:
  - type: http
    method: post
    fetchFn: 
    params:
      - key: name
      - key: memo
    display: form
```

### Set input format

```yaml
params:
  - key: memo
    format: textarea
```

### Group and styling form

```yaml
params:
  - key: name
    group: 1
  - key: is_guest
    group: 1
  - key: phone
    style:
      width: 100%
  - key: email
display: form
formOptions:
  firstLabelWidth: 120px
  labelWidth: 100px
```

### Validate input

브라우저 단에서 input 검증(validate) - 입력 값 가이드 역할

```yaml
params:
  - key: email
    validateFn: |
      const value = param.value?.trim()
      const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/

      if (!value) {
        return '필수 입력 항목'
      }
      if (!emailRegex.test(value)) {
        return '이메일 형식이 아님'
      }
      return '';
```

### Disable button by condition with JS

값이 알맞지 않을때 제출 버튼 비활성화

```yaml
params:
display: form
submitButton: 
  disabledFn: |
    const email = params.find(e => e.key == 'email')
    const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/

    if (!email.value || !emailRegex.test(email.value.trim())) {
      return true
    }
    return false
```
