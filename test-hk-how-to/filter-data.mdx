### Filter data with input field

```yaml
blocks:
  - type: http
    method: GET
    fetchFn: |
      const q = name || 'all'
      return [
        { id: 1, keyword: q, result: 'mock-1' },
        { id: 2, keyword: q, result: 'mock-2' }
      ]
    params:
      - key: name
        placeholder: 검색어 입력
```

### Set input format and component

```yaml
params:
  - key: date
    format: date
  - key: select
    dropdown:
      - Brut
      - Rosé Brut
```

### Styling filter

```yaml
display: form
# display: col
params:
  - key: name
  - key: email
```

### Filter multiple blocks

```yaml
params:
  - key: name
blocks:
  - type: http
  - type: http
```

### Search table in client-level

```yaml
type: http
searchOptions:
  enabled: true
```

### Filter column in client-level

```yaml
type: http
columns:
  name:
    filterOptions:
      enabled: true
```

### Paginate table in client-level

```yaml
type: http
paginationOptions:
  enabled: true
```

### Sorting table

```yaml
type: http
sortOptions:
  enabled: true
  initialSortBy:
    field: created_at
    type: desc # desc, asc
```
