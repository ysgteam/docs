---
title: "Make a chart"
description: "여러 차트를 통해 데이터를 시각화합니다."
---

### Basic chart

```yaml
blocks:
  - type: http
    chartOptions:
      type: bar # line, doughnut, polarArea
      x: type
      y: count
    showButtonWithResult: true
```

### Set multiple y-axis

```yaml
chartOptions:
  type: bar
  x: date
  y: 
    - company
    - user
```

### Change background color of chart

```yaml
chartOptions:
  type: bar
  x: date
  y: 
    - company
    - user
  backgroundColor:
    company: "RoyalBlue"
    user: "#0D6EFD"
```
