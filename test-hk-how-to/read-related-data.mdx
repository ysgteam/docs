title: "Read related data"
description: "데이터 조회후 관련 내용을 추가로 보여주는 방법입니다."
---

### Get related data in modal with parameter

`modals`, `openModal`

- 모달로 조회

```yaml
type: http
columns:
  id:
    openModal: modal1-:id
modals:
  - path: modal1-:id
    blocks:
      - type: http
        params:
          - key: id
            valueFromRow: id
```

### Get data in tab with parameter

`tabOptions` , `tabs`

- 탭에서 조회

```yaml
type: http
tabOptions:
  tabs:
    - name: 1
      blocks:
        - type: http
          params:
            - key: id
              valueFromRow: id
```

### Reload data after updating parameter of another block

`updateParams`

- 다른 블록에서 조회(레이아웃)

```yaml
- type: left
  columns:
    id:
      updateParams:
        id2: "{{id}}"
- type: right
  params:
    - key: id2
```

### Send parameter to another page

`valueFromSearch`

페이지 이동해서 조회

```yaml
columns:
  id:
    openPage: value-from-search-:id

---
pages:
  - path: value-from-search-:id
    blocks:
      - type: http
        params:
          - key: id
            valueFromSearch: id
```
