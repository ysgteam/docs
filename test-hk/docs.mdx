GET API 호출

```yaml
blocks:
  - type: http
    method: GET
    fetchFn: |
      return [
        { id: 1, name: 'tom', point: 70 },
        { id: 2, name: 'jerry', point: 80 }
      ];
```

GET 응답 데이터는 기본적으로 테이블로 표기됩니다.

데이터 표현을 바꾸고 싶을때

데이터 테이블 꾸미기

```yaml
blocks:
  - type: http
    method: GET
    fetchFn: |
      return [
        { id: 1, name: 'tom', point: 70 },
        { id: 2, name: 'jerry', point: 80 }
      ];
    tableOptions:
      width: 120px
```

차트

```yaml
blocks:
  - type: http
    method: GET
    fetchFn: |
      return [
        { id: 1, name: 'tom', point: 70 },
        { id: 2, name: 'jerry', point: 80 }
      ];
    chartOptions:
      type: bar
      x: name
      y: point
```

HTML 템플릿

```yaml
blocks:
  - type: http
    method: GET
    fetchFn: |
      return [
        { id: 1, name: 'tom', point: 70 },
        { id: 2, name: 'jerry', point: 80 }
      ];
    template: |
```

입력 폼을 만들고 싶을때

```yaml
blocks:
  - type: http
    method: POST
    fetchFn: |
    params:
      - key: name
      - key: point
```

폼 꾸미기

```yaml
    params:
      - key: name
        group: 1
      - key: point
        group: 2
      - key: status
        group: 2
    display: form
    formOptions:
```

입력 포맷 설정

```yaml
    params:
      - key: name
        group: 1
      - key: point
        group: 2
        format: number
      - key: status
        group: 2
        dropdown:
          - active
          - inactive
    display: form
    formOptions:
```
